# Start Frontend Development Server

Starts the React + Vite frontend development server.

```bash
npm run dev
```

Server runs on **http://localhost:5999**

## Project Info
- **Framework**: React 18 + Vite 6 + TailwindCSS
- **Port**: 5999
- **Entry**: `src/main.jsx`
